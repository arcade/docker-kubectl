## Install

`docker pull koudaiii/kubectl`

## Usage

* `docker run -it -v ~/.kube/config:/root/.kube/config koudaiii/kubectl`
* `docker run -it -v ~/.kube/config:/root/.kube/config koudaiii/kubectl version`
